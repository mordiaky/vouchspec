- [x] Initial work uses only public, synthetic, or authorized artifacts.
- [x] No credentials or sensitive personal data are required.

- [x] Paid placement is separated from evidence ranking.
- [ ] Working name conflict search completed.
- [ ] Artifact license handling verified on the real sample.
- [ ] HTTP and MCP inputs have explicit size and path-containment limits.
- [ ] Independent security/API review completed.
- [ ] Public terms/privacy/methodology prepared before external launch.
- [ ] Jurisdiction-specific obligations reviewed before accepting payment.
