# Fulfillment

The Stage B sandbox now completes the constrained public-source flow: immutable GitHub commit
and subdirectory intake, byte/blob freeze, no-egress static worker, constrained exact-byte
DSSE signing, complete cost ledger, and signed delivery. It never imports or executes artifact
content.

See [the operating and security boundary](../docs/stage-b-operations.md) and the checked-in
[non-commercial sandbox proof](stage-b-sandbox-proof.json). Live checkout and public intake
remain disabled until the explicit live gates in that document pass.
