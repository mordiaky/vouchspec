# Fulfillment

The public Stage B sandbox completes the constrained public-source flow: authenticated tenant
and order creation, exact test-USDC x402 settlement on Base Sepolia, immutable GitHub commit
and subdirectory intake, byte/blob freeze, a no-egress static worker, a separate no-egress
signer, complete cost ledger, and signed delivery. It never imports or executes artifact
content.

See [the operating and security boundary](../docs/stage-b-operations.md), the checked-in local
[non-commercial sandbox proof](stage-b-sandbox-proof.json), and hosted owner-excluded proof in
the operations document. The sandbox is public and self-service. It has no human checkout and
does not count as revenue or adoption. Mainnet remains fail-closed until the explicit live gates
in that document pass.
